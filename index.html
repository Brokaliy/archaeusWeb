What makes it different

Power without the learning curve

Archaeus keeps everything that makes Arch Linux great — rolling releases, AUR access, bleeding-edge packages — and wraps it in a UI that makes sense.

Archaeus Control Center

Install apps, switch desktop environments, tweak appearance and keybinds — all from a clean Electron GUI. No config file editing needed.

Hyprland by default

Ships with a fully configured Hyprland compositor — rounded corners, blur, fluid animations, and a warm Waybar status bar ready out of the box.

Coffee-shop aesthetic

Every pixel follows a warm dark palette — charcoal backgrounds, cream text, amber accents. Beautiful enough to use in public.

Full AUR access

Under the hood it's still Arch. yay and pacman are fully available. Power users have everything they've always had.

Switch DEs visually

Prefer KDE Plasma or GNOME? Select it in the Desktop tab and Archaeus installs, configures, and sets it as default — one click.

First-boot experience

The ACC launches automatically on first boot, guiding you through app installation and personalization before you ever touch a terminal.

The Control Center

Your system, your way — visually

The Archaeus Control Center (ACC) is the core differentiator. It's an Electron app that wraps every common system task in a clean interface.

  • Package manager GUI

    Grid of common apps. Click to install via yay. Search anything in the AUR.

  • Desktop switcher

    Live preview thumbnails of Hyprland, KDE Plasma, and GNOME.

  • Appearance editor

    Accent color picker, wallpaper gallery, font selector.

  • Visual keybind editor

    Set Hyprland keybinds without touching hyprland.conf.

Philosophy

The terminal isn't
going anywhere.

Archaeus doesn't hide the terminal — it makes it optional. Every visual action the ACC takes is a real shell command running under the hood. Open a terminal and you'll see exactly what it did. Power users lose nothing; everyone else gains everything.
