Getting started

Install in 4 steps

1
Flash the ISO to a USB drive

Use Balena Etcher or dd to write the ISO to a USB drive (8 GB+).

sudo dd if=archaeus-0.1.0-x86_64.iso of=/dev/sdX bs=4M status=progress
2
Boot from USB

Enter your BIOS/UEFI and set the USB as the first boot device. Secure Boot should be disabled. You'll land in a live Archaeus environment.

3
Run the installer

The live environment includes the archaeus-install TUI installer. It handles partitioning, user creation, and bootloader setup.

archaeus-install
4
First boot — ACC launches automatically

On your first boot the Archaeus Control Center opens automatically, walking you through app installation and personalisation. You're done.
