Features

A full Arch Linux system with a five-panel control center. Every power-user tool is still there.

ACC — Packages

42 apps. One click each.

Six categories, 42 curated apps. Click to install or remove — ACC streams yay -S or yay -Rns output live in the log. The search bar hits the full AUR.

  • Browsers: Firefox, Chromium, Brave, Tor Browser, Vivaldi
  • Chat: Discord, Slack, Telegram, Signal, Element
  • Media: Spotify, VLC, mpv, OBS, Kdenlive, GIMP, Inkscape, Audacity
  • Dev: VS Code, Neovim, Zed, JetBrains Toolbox, Docker, Podman, Insomnia
  • Gaming: Steam, Lutris, Heroic, Bottles, Mangohud
  • Tools: Thunar, Ark, Bitwarden, Obsidian, LibreOffice, Flameshot, Remmina

ACC — Desktop

Switch DEs without reinstalling.

Reads the current DE from ~/.config/archaeus/default-de, shows three preview cards, installs packages, updates SDDM. Revert any time from the login screen session picker.

  • Hyprland — default, Wayland-native tiling compositor
  • KDE Plasma — full-featured, highly customisable
  • GNOME — clean, modern, great touchpad workflow
  • Your home directory is never touched on switch

ACC — Appearance

7 accent colors. Your wallpaper. Your font.

Amber, Rose, Sage, Steel, Lavender, Teal, Peach. Changes apply live to Hyprland borders, Waybar, and the ACC itself. Wallpaper updates hyprpaper.conf and restarts hyprpaper.

  • 7 named accent presets with live preview
  • Wallpaper: local file picker, updates hyprpaper, instant apply
  • Fonts: Inter, JetBrains Mono, Noto Sans, Geist

ACC — Startup

Autostart managed with toggles.

Scans ~/.config/autostart/ for .desktop files. Toggling off sets Hidden=true — no files deleted, fully reversible.

  • Reads real .desktop autostart entries
  • Toggle is instant — no restart needed
  • Fully reversible — just toggle back on

ACC — Keybinds

All your binds, parsed live.

Reads your live hyprland.conf and displays every bind = ... line as a readable table. The default Archaeus config has full coverage out of the box.

  • Super/Mod key for all window management
  • vim-style focus: h/j/k/l alongside arrow keys
  • XF86 media, volume, brightness keys pre-wired
  • Workspaces 1–10, Super+Shift to move windows

Under the hood

The stack

Base
Arch Linux

Rolling release, 130 packages, archiso build. BIOS + UEFI. TUI installer with disk partitioning, user creation, and GRUB.

Compositor
Hyprland

12px rounding, 8px 2-pass blur, #C8A96E amber borders, smooth bezier animations. Waybar: clock, audio, bluetooth, network, battery, ACC icon.

Control Center
Electron ACC

900x650 frameless Electron window. IPC handlers: yay wrapper, DE switching, hyprpaper, autostart .desktop files, hyprland.conf bind parser.

Terminal
Kitty + zsh

JetBrains Mono Nerd Font 13px, 92% bg opacity, 10k line scrollback. Zsh with autosuggestions, syntax-highlight, Starship prompt in amber theme.
