Installation

Archaeus installs from a bootable ISO. The process takes around 10–15 minutes on a modern machine.

1. Create a bootable USB

Flash the ISO to a USB drive (8 GB minimum). You can use Balena Etcher on any OS, or dd on Linux/macOS:

sudo dd if=archaeus-0.1.0-x86_64.iso of=/dev/sdX bs=4M status=progress && sync
Warning: Replace /dev/sdX with your USB drive. Double-check with lsblk first — this command will overwrite anything on the target device.

2. BIOS settings

  • Disable Secure Boot
  • Set boot mode to UEFI (recommended) or Legacy/CSM
  • Set USB as the first boot device

3. Run the installer

From the live environment, open a terminal and run:

archaeus-install

The TUI installer will guide you through:

  • Disk partitioning (auto or manual)
  • Locale and timezone selection
  • Username and password setup
  • Bootloader installation (systemd-boot or GRUB)

First Boot

On your first login, Hyprland starts and the Archaeus Control Center opens automatically in setup mode. It walks you through:

  1. Choosing apps to install from a curated list
  2. Setting your accent color and wallpaper
  3. Optionally switching to KDE Plasma or GNOME

The setup wizard only appears on the first login. After that, you can open ACC at any time with SUPER + A.

Tip: The live environment during installation is a full preview of Archaeus. Explore before you commit — nothing is written to disk until you run archaeus-install.

ACC — Packages

The Packages tab provides a grid of popular applications. Click any app to install it — ACC runs yay -S <package> in the background and shows a progress indicator.

For anything not in the grid, use the search bar. It queries the AUR and pacman repositories and shows results instantly.

Under the hood: ACC uses yay as the AUR helper. You can always check what was installed by running yay -Qe in a terminal.

ACC — Desktop

Switch between Hyprland, KDE Plasma, and GNOME from the Desktop tab. Archaeus installs the required packages and sets the new session as the SDDM default. The change takes effect on next login.

Note: Switching DEs does not remove the previous one. You can switch back at any time, or log in to an older session from the SDDM login screen by selecting a different session.

ACC — Appearance

The Appearance tab controls:

  • Accent color — updates Hyprland border color, Waybar, and ACC itself
  • Wallpaper — picks from a curated gallery or browse local files
  • Font — system-wide UI font (Inter, Geist, or custom)
  • Corner radius — Hyprland window rounding (0–20px)
  • Blur — toggle and strength

ACC — Keybinds

Visual editor for Hyprland keybinds. Click any action row, press a key combination, and ACC writes the binding to ~/.config/hypr/hyprland.conf and reloads the config.

ACC — Startup

The Startup tab lists autostart entries. Toggle any app on or off — ACC writes to ~/.config/hypr/hyprland.conf exec-once entries.

Hyprland Configuration

The Archaeus Hyprland config lives at ~/.config/hypr/hyprland.conf. While the ACC handles most settings visually, you can also edit it directly. 

# Reload config without restarting
hyprctl reload

# Check active config
hyprctl getoption general:gaps_out

Workspaces

Hyprland uses numbered workspaces (1–10). Navigate with:

  • SUPER + 1–9 — switch to workspace
  • SUPER + SHIFT + 1–9 — move focused window to workspace
  • SUPER + scroll — cycle through workspaces
  • Waybar workspace buttons — click to jump

Using the AUR

Archaeus ships with yay as the AUR helper. You can use it directly in the terminal — ACC is just a wrapper around it. 

# Install a package
yay -S package-name

# Update everything
yay -Syu

# Search
yay -Ss keyword

# Remove a package
yay -Rns package-name

Extending ACC

ACC is an open-source Electron app. The source lives in ~/.local/share/archaeus/acc/. You can add new panels by creating a new view file and registering it in src/renderer/views/index.js.

Coming soon: A plugin API for third-party ACC panels is planned for a future release.
