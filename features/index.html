Features

A full Arch Linux system with a control center that makes it accessible — without sacrificing a single power user feature.

Core feature

Archaeus Control Center

The ACC is a native Electron application that ships as part of the OS. It provides a GUI layer over the most common system administration tasks — no terminal knowledge required.

  • Install packages from a curated grid or search the entire AUR
  • Switch between Hyprland, KDE Plasma, and GNOME with one click
  • Visual keybind editor that writes to hyprland.conf
  • Appearance panel for accent color, wallpaper, fonts, and corner radius
  • Startup manager to toggle autostart applications

Default compositor

Hyprland, pre-configured

Hyprland is a Wayland compositor known for smooth animations and a highly customisable tiling layout. Archaeus ships a complete, opinionated config that looks great on first boot — no config editing required.

  • Rounded corners and subtle window blur
  • Smooth animations (fade, slide, workspaces)
  • Custom Waybar with clock, workspace indicators, media info
  • Sensible default keybinds (overrideable in ACC)
  • SDDM login screen themed to match
~/.config/hypr/hyprland.conf
# Archaeus default — warm theme
 
general {
  gaps_in = 5
  gaps_out = 12
  border_size= 1
  col.active_border = rgba(c4874aff)
}
 
decoration {
  rounding = 12
  blur {
    enabled = true
    size = 8
  }
}
Desktop environments

Switch DEs without reinstalling

Not a Hyprland fan? The Desktop tab in ACC lets you install and switch to KDE Plasma or GNOME. Archaeus installs the packages, sets the session default in SDDM, and you're done.

  • Hyprland — fast, tiling, Wayland-native
  • KDE Plasma — feature-rich, familiar, highly customisable
  • GNOME — clean, modern, touch-friendly workflow
  • Live preview thumbnails before switching
  • Revert at any time without reinstalling the OS

Keybinds

Visual keybind editor

Hyprland keybinds are normally configured by hand in hyprland.conf. The ACC Keybinds tab abstracts this — click an action, press your desired combo, and it writes the config for you.

  • All default Archaeus actions listed and re-mappable
  • Changes apply without restarting Hyprland
  • Conflict detection prevents binding the same combo twice
  • Export / reset to defaults at any time

Under the hood

The tech stack

Base
Arch Linux

Rolling release, minimal base, full AUR access via yay. Built with archiso into a bootable ISO.

Compositor
Hyprland

Wayland compositor with fluid animations, tiling layouts, blur, and rounded corners. Custom Waybar status bar.

Control Center
Electron ACC

Custom Electron app providing GUI over package management, DE switching, appearance, and keybinds.

Terminal
Kitty + zsh

Kitty terminal with GPU acceleration, coffee-shop color theme, and Starship prompt for a clean shell experience.
